# MCP Tools

Install with 

```bash
uv tool install .
```

Then run

```bash
claude mcp add ollama-mcp -s user -- ollama-mcp
```

substituting `ollama-mcp` for the appropriate tool.

To install for opencode run
```bash
opencode mcp add
```
and follow the hints.

## Ollama MCP

Allows task to be offloaded to different ollama models.
